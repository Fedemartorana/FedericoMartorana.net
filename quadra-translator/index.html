Traduzione & Builder

Italiano (parole / frasi)

Proposte AI (se parola non trovata nel vocabolario)

Se una parola non esiste ancora, l’API genera proposte rispettando i token ammessi e l’ordine consigliato. Accetta & salva aggiorna il vocabolario in Cloudflare D1 (persistente, condivisibile tra device).
Quadra (glifi + gloss)

Builder frase (composizione manuale)

Costruisci parole Quadra e concatenale in una frase. Ogni parola è una sequenza di glifi (+). Le parole sono separate da /. Negazione: anteponi ! al token da negare (!PRESENTE) → inserisce “assenza” dentro quel glifo.

Vocabolario (Italiano → Quadra) • Cloud + cache locale

Salva voce scrive su Cloudflare D1 (backend). In più viene mantenuta una cache locale per listare velocemente le voci accettate da questo browser.
Formato: TOKEN+TOKEN+TOKEN+TOKEN e più parole con /. Negazione con !TOKEN.

Voci in cache locale (browser)

Questa lista è solo la cache di questo browser. La fonte verità è il backend. Per verificare una singola voce usa Cerca voce (cloud).

Legenda glifi

Set minimo implementato: entità, agenzia, stato, tempo + relazione/inizio/fine. La lingua cresce aggiungendo voci al vocabolario, non nuovi segni.

Come “impara”

  • Traduzione: chiamata API /translate (AI + lookup DB).
  • Accetta & salva: API /vocab/add (persistente in Cloudflare D1).
  • Cache locale: memorizza solo le voci accettate da questo browser (per comodità UI).
